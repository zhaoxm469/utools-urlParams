# queryParams-uToolsPlugin 

* queryParam 与 JSON互转的 utools插件

### 仓库地址

* https://github.com/zhaoxm469/utools-urlParams


### 插件说明

* utools输入框粘贴文本自动识别 queryParams 和 JSON   
* 按照选项进行转换即可
